# Como contribuir com sua conta ElevenLabs

Precisamos de áudios sintéticos para um projeto de pesquisa em detecção de deepfake de voz (ExACTa-PUC). Você pode ajudar em menos de 5 minutos — só precisa criar uma conta gratuita e nos enviar dois dados.

---

## O que você precisa fazer

### 1. Criar conta gratuita no ElevenLabs

Acesse [elevenlabs.io](https://elevenlabs.io) e crie uma conta com seu e-mail.
Não é necessário cartão de crédito.

### 2. Pegar sua API Key

1. Clique no seu avatar (canto inferior esquerdo)
2. Vá em **API Keys**
3. Clique em **Create API Key**, dê qualquer nome
4. Copie a chave gerada — ela começa com `sk_...`

### 3. Escolher uma voz

1. No menu lateral, clique em **Voices**
2. Escolha qualquer voz da lista **premade**
3. Clique nela e copie o **Voice ID** (sequência de letras e números)

### 4. Enviar para o responsável

Mande os dois dados por mensagem:

```
API Key:  sk_...
Voice ID: ...
```

---

Pronto. O resto é feito aqui. Sua cota de 10.000 caracteres gratuitos por mês será usada para gerar ~250 frases em português para o dataset.
